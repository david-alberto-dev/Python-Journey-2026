# 📐 ShapeMath: Area & Volume Calculator
A comprehensive geometric calculator that handles both 2D and 3D shapes using advanced function mapping for clean and efficient execution.

# ⚡ What does it do?
The user is asked to enter whether they wish to calculate the area of a 2D shape, or the volume of a 3D one. Based on their choice,
the program will provide a list with available shapes for the asked dimension. Then it uses a dictionary-based dispatch system
to link user choices to their respective functions, avoiding long if-elif chains. 
Finally, a dedicated function exists to ensure inputs are always positive numeric values.

# 🛠️ Tech Stack
**Language:** Python v3.14.0

**Modules:** math (for π)

**Concepts:** Higher-order functions (storing functions in dictionaries), argument unpacking, error handling.
