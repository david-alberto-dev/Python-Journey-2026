# 💰 Expense Tracker
A smart personal finance CLI tool that helps users manage their daily spending and track their remaining budget in real-time.

## 📖 Description
The Expense Tracker is designed to provide clarity on personal finances. 
It allows users to set an initial budget and then log individual expenses by name, category, and price.

The application automatically calculates the remaining balance after every addition or removal, 
ensuring that the user's financial status is always up to date.

## ⭐️ Features
- **Budget Management:** Set a starting budget and see it update dynamically.

- **Detailed Logging:** Each expense tracks the item name, category, and cost.

- **Refund Logic:** Removing an expense automatically "refunds" the amount back to the current budget.

- **Data Integrity:** Uses a list of dictionaries to maintain structured and accessible data.

- **Error Handling:** Protects against invalid inputs (e.g., entering text instead of prices).

## 🧰 Tech Stack
- **Language:** Python v3.14+
- **Focus:** Data Structures
