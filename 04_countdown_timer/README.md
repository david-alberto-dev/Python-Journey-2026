# ⏳ Terminal Countdown Timer
A simple CLI countdown tool built with Python, featuring real-time terminal updates and robust input validation.

This project was a deep dive into Buffer Management and Terminal Control Characters, ensuring a professional look without cluttering the console history.

## ✨ Features
- **Dynamic In-Place Updates:** Time ticks in the same spot using \r and flush=True, preventing terminal "scrolling".


- **Smart Input Handling:** Prevents the user from entering negative values or anything that is not a number.

## 🧰 Tech Stack
- **Language:** Python v3.14+

- **Core Module:** Time

## 📥 Installation & Usage
1. Clone the repository:
```bash
git clone https://github.com/david-alberto-dev/Python-Journey-2026.git
```
2. Navigate to the directory:
```bash
cd Python-Journey-2026/04_countdown_timer/
```
3. Run the program:
```bash
python3 countdown_timer.py 
```
