# 🔢 Basic Arithmetic Calculator
The first project of my journey, focusing on core Python logic, control flow, user input, error handling, and basic mathematical operations.

# ⚡ What does it do?
- Supports basic operations such as addition, subtraction, multiplication, division, and exponentiation.
- Handles ValueError (invalid numbers) and ZeroDivisionError (prevents the app from crashing when dividing by zero).
- Allows infinite calculations without restarting the script.

# 🛠️ Tech Stack
**Language:** Python v3.14.0

# 🚀 Usage
```Bash
git clone https://github.com/david-alberto-dev/Python-Journey-2026/tree/main/01-Simple-Calculator/

cd 01-Simple-Calculator

python3 calculator.py
```
