# ✈️ Python Journey 2026
This repository contains all miniprojects I built throughout my journey and represents my progress.

I started learning Python back in 2021 but I didn't remain consistent. I was stuck in tutorial hell, a lot of confusion and fear of "not being prepared enough" before actually building anything. However, I decided to return in 2026 with a different approach: learning by building.

Each one contains its own README.md file, including installations and usage instructions.
